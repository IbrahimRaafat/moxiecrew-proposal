## 2. UX/UI ANALYSIS TOOLS

### Primary Tools

| Tool | Free/Paid | What It Reveals | Real Metrics |
|------|-----------|-----------------|--------------|
| **Hotjar** | Free/Paid | Heatmaps, click maps, session recordings | WHERE users click most |
| **Crazy Egg** | Paid | Scroll maps, confetti clicks | User attention patterns |
| **Clarity (Microsoft)** | Free | Session recordings, heatmaps | 100% free alternative to Hotjar |
| **Google Analytics 4** | Free | User journeys, behavior flow, drop-off points | Funnel abandonment %, bounce rate |
| **LogRocket** | Paid | Session replay + console logs | Frontend error visibility |

### Key Metrics to Measure
- Homepage bounce rate % (by device: mobile vs desktop)
- Product page → Cart conversion funnel drop-off %
- Scroll depth (how far users scroll on key pages)
- Click heatmaps (which CTAs are ignored, which are clicked)
- Session recording insights (user struggles, confused clicks, dead-end navigation)
- Form abandonment rates (checkout, newsletter signup)
- Mobile UX friction points (touch targets, readability, navigation)
- Error frequency (404s, JS errors, network issues)

### Data Collection Timeline
- **Time estimate**: 3-5 hours
- **Tools needed**: Hotjar (free trial), Clarity (free), GA4 (free)
